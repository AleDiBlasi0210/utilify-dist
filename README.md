# utilify-dist

Repository **pubblico** di distribuzione di Utilify (il codice sorgente resta privato).

- `manifest.v1.json` — elenco degli strumenti letto dall'app all'apertura dello Store.
- I binari dei pacchetti (fasi successive) saranno pubblicati come asset dei **Releases**.

Non modificare a mano: il manifest è generato dal repo sorgente.
